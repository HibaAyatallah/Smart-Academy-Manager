# Assets requis pour Finatech Group (Portail)

Ce fichier répertorie toutes les images et logos officiels nécessaires à la transformation du projet en portail institutionnel.
Merci de remplacer les fichiers SVG (*placeholders*) actuels par vos propres ressources en respectant strictement les noms, formats, et dimensions indiqués.

---

### 1. Logos & Identité visuelle
Dossier : `frontend/public/images/logos/`

| Fichier attendu      | Format | Dimensions recommandées | Utilisation |
| :---                 | :---   | :---                    | :---        |
| **logo_color.svg**   | SVG / PNG | ~ 400x100 px (ratio horizontal) | En-tête du portail (fond clair) |
| **logo_white.svg**   | SVG / PNG | ~ 400x100 px (ratio horizontal) | Pied de page ou menus sombres |
| **logo_icon.svg**    | SVG / PNG | 100x100 px (carré)              | Micro-interactions, espace très réduit |

### 2. Favicon
Dossier : `frontend/public/`

| Fichier attendu | Format | Dimensions recommandées | Utilisation |
| :---            | :---   | :---                    | :---        |
| **favicon.ico** | ICO    | 32x32 px ou 64x64 px    | Icône de l'onglet du navigateur |

---
**Note technique :**
Pour des raisons de performance, il est fortement recommandé de compresser les photographies en format WEBP. Si vous utilisez du WEBP ou PNG, veillez simplement à adapter l'extension dans le code source HTML le moment venu.
